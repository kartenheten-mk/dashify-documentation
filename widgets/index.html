    Widgets

    Dashify offers a rich set of widgets to visualize and interact with your data. Each widget is configured within the widgets array of a dashboard.

    Overview

    Widgets are categorized into:

    • Primary Widgets: map (central view), table (data grid).
    • Sidebar Widgets: counter, area, length, statistics, linechart, areachart, piechart, populationpyramid (displayed in the right sidebar).

    Widget Configuration

    Each widget object in the widgets array follows this general structure:

    {
  "type": "widget_type",
  "title": "Widget Title",
  "dataSource": "datasource_id", // ID matching the dashboard's dataSource
  "options": {
    // Widget-specific options
  }
}

    Map Widget (map)

    Displays GeoJSON features on an interactive Leaflet map. Supports clustering, popups, multiple base layers, and layer switching.

    Options (options)

    Property Type Default Description
    center [number, number] [62.0, 15.0] Initial map center [lat, lon].
    zoom number 5 Initial zoom level.
    markerColor string #3b82f6 Color for point markers (hex code).
    mapSource string - Key of the base map source defined in mapSources.
    layers string[] - Optional layer override for WMS sources.
    additionalLayers (string \| object)[] - Extra layers appended to the map source defaults (drawn on top).
    showLayerControl boolean false Enable the layer switcher control.
    showColorPicker boolean false Enable the color picker control.
    queryable boolean true Enable click interactions (popups + selection).
    enableClustering boolean false Enable clustering for point features.
    clusterAllGeometries boolean false Cluster all geometry types (including polygons) using centroids.
    clusterRadius number 50 Max cluster radius in pixels.
    disableClusteringAtZoom number - Zoom level to disable clustering and show individual markers.
    spiderfyOnMaxZoom boolean true When clicking a cluster at max zoom, spiderfy markers instead of zooming.
    showCoverageOnHover boolean false Show the bounds of a cluster on hover.
    zoomToBoundsOnClick boolean true Zoom to bounds of cluster when clicked.
    animate boolean true Enable/disable cluster animations.
    chunkedLoading boolean true Load markers in chunks for better performance.
    removeOutsideVisibleBounds boolean true Remove markers outside viewport for better performance.
    showSingleFeatures boolean true Show individual markers when not clustered. Set to false for density-only views.
    hideAtZoom number - Hide all features when zoomed in beyond this level.

    Example

    {
  "type": "map",
  "title": "Main Map",
  "dataSource": "my-data",
  "options": {
    "center": [59.3293, 18.0686],
    "zoom": 10,
    "enableClustering": true,
    "clusterAllGeometries": true,
    "markerColor": "#e74c3c",
    "mapSource": "osm",
    "showLayerControl": true
  }
}

    Table Widget (table)

    Displays a sortable, paginated table of feature properties. Supports column filtering and searching.

    Options (options)

    Property Type Default Description
    columns string[] All List of property keys to display as columns.
    sortable boolean true Enable column sorting.
    pageSize number 50 Number of rows per page.

    Example

    {
  "type": "table",
  "title": "Feature List",
  "dataSource": "my-data",
  "options": {
    "columns": ["id", "name", "category", "date_created"]
  }
}

    Length Widget (length)

    Calculates the total length of line features.

    Options (options)

    Property Type Default Description
    label string - Label for the length value.
    unit 'm' \| 'km' \| 'mil' 'km' Unit of measurement.
    precision number 2 Number of decimal places.
    groupBy string - Attribute to group the length by.
    filterValues string[] - Only include these specific groupBy values.

    Example

    {
  "type": "length",
  "title": "Road Network",
  "dataSource": "roads",
  "options": {
    "unit": "km",
    "groupBy": "road_type"
  }
}

    Statistics Widget (statistics)

    Displays multiple calculated statistics (sum or count) based on feature attributes.

    Options (options)

    Property Type Default Description
    statistics StatisticsItem[] - Array of statistics to calculate.
    groupBy string - Optional attribute to group/pivot by.
    groupByDate DateGroupingConfig - Optional date grouping configuration.
    filterValues string[] - Optional: only include these groupBy values.
    uniqueBy string - Optional: deduplicate features by this attribute before calculating statistics.

    StatisticsItem Structure

    Property Type Description
    id string Unique ID.
    label string Display label.
    attribute string Feature property to operate on (* for all).
    operation 'sum' \| 'count' Aggregation method.
    unit string Suffix for the value (e.g., "pcs").
    precision number Number of decimal places.

    DateGroupingConfig Structure

    Property Type Description
    dateField string The date field to use for grouping.
    intervalDays number Number of days per interval.
    maxIntervals number Optional: Maximum number of intervals to show.

    Example

    {
  "type": "statistics",
  "title": "Project Stats",
  "dataSource": "projects",
  "options": {
    "groupByDate": {
      "dateField": "created_at",
      "intervalDays": 7,
      "maxIntervals": 4
    },
    "statistics": [
      {
        "id": "total_cost",
        "label": "Total Cost",
        "attribute": "budget",
        "operation": "sum",
        "unit": "SEK"
      }
    ]
  }
}

    Line Chart Widget (linechart)

    Visualizes data trends over time.

    Options (options)

    Property Type Default Description
    dateField string - The timestamp field to use for the X-axis.
    valueField string - Optional field to sum (defaults to count).
    operation 'count' \| 'sum' \| 'mean' \| 'median' \| 'max' \| 'min' 'count' Aggregation method.
    defaultGranularity 'day' \| 'week' \| 'month' \| 'year' 'week' Default time interval.
    groupBy string - Optional attribute to create multiple lines.
    label string - Y-axis label.
    lineColor string '#3b82f6' Line color.
    showDots boolean true Show data points.
    compact boolean false Compact mode for sidebar.
    uniqueBy string - Optional: deduplicate features by this attribute.
    groupColors string[] - Optional: custom color palette for groups.
    filterField string - Optional: attribute field to filter on.
    filterValues string[] - Optional: only include features where filterField matches one of these values.
    yAxisAutoScale boolean false Optional: auto-scale Y-axis to data range.

    Example

    {
  "type": "linechart",
  "title": "Applications per Month",
  "dataSource": "applications",
  "options": {
    "dateField": "submission_date",
    "defaultGranularity": "month",
    "groupBy": "status",
    "groupColors": ["#3b82f6", "#ef4444", "#10b981"]
  }
}

    Area Chart Widget (areachart)

    Visualizes data trends over time as filled areas. Supports stacking for grouped data and configurable curve interpolation. Inherits all options from linechart.

    Options (options)

    Property Type Default Description
    dateField string - The timestamp field to use for the X-axis.
    valueField string - Optional field to aggregate (defaults to count).
    operation 'count' \| 'sum' \| 'mean' \| 'median' \| 'max' \| 'min' 'count' Aggregation method.
    defaultGranularity 'day' \| 'week' \| 'month' \| 'year' 'week' Default time interval.
    groupBy string - Optional attribute to create multiple areas.
    stacked boolean false Stack areas on top of each other (requires groupBy).
    curveType 'monotone' \| 'linear' \| 'step' \| 'stepBefore' \| 'stepAfter' \| 'natural' 'monotone' Interpolation curve type. Use linear for straight lines between points.
    label string 'Antal' Y-axis label.
    lineColor string '#3b82f6' Area stroke/fill color (single area mode).
    showDots boolean true Show data points.
    compact boolean false Compact mode for sidebar.
    uniqueBy string - Optional: deduplicate features by this attribute.
    groupColors string[] - Optional: custom color palette for groups.
    filterField string - Optional: attribute field to filter on.
    filterValues string[] - Optional: only include features where filterField matches one of these values.
    yAxisAutoScale boolean false Auto-scale Y-axis to data range instead of starting from 0.

    Example (Basic)

    {
  "type": "areachart",
  "title": "Registrations Over Time",
  "dataSource": "registrations",
  "options": {
    "dateField": "created_date",
    "defaultGranularity": "month",
    "lineColor": "#8b5cf6",
    "curveType": "linear"
  }
}

    Example (Stacked by Group)

    {
  "type": "areachart",
  "title": "Applications by Municipality",
  "dataSource": "applications",
  "options": {
    "dateField": "submission_date",
    "groupBy": "municipality",
    "stacked": true,
    "curveType": "linear",
    "defaultGranularity": "week",
    "groupColors": ["#4ade80", "#60a5fa", "#f472b6"]
  }
}

    Pie Chart Widget (piechart)

    Displays distribution of data based on a grouping attribute.

    Options (options)

    Property Type Default Description
    groupBy string - Attribute to group by (creates segments).
    operation 'count' \| 'sum' \| 'length' \| 'area' 'count' Aggregation method.
    valueField string - Field to sum (if operation is sum).
    unit string - Optional: unit for length/area.
    precision number - Optional: decimal places for length/area.
    colors string[] - Optional: custom color palette.
    showLabels boolean true Show percentage labels on segments.
    compact boolean false Compact mode for sidebar.
    uniqueBy string - Optional: deduplicate features by this attribute.
    filterValues string[] - Optional: only include these groupBy values.
    innerRadius number 0 Percentage (0-100) for donut style.
    showLegend boolean true Whether to show the legend.

    Example

    {
  "type": "piechart",
  "title": "Status Distribution",
  "dataSource": "tasks",
  "options": {
    "groupBy": "status",
    "innerRadius": 50,
    "colors": ["#3b82f6", "#ef4444", "#10b981"]
  }
}

    Population Pyramid Widget (populationpyramid)

    Specialized chart for displaying age and gender distribution.

    Options (options)

    Property Type Default Description
    genderField string - Field containing gender data.
    ageField string - Field containing age data.
    maleValue string "M" Value representing male in genderField.
    femaleValue string "K" Value representing female in genderField.
    maleLabel string "M√§n" Display label for males.
    femaleLabel string "Kvinnor" Display label for females.
    ageInterval number 5 Size of age buckets.
    maxAge number 100 Max age before "X+" group.
    maleColor string "#3b82f6" Color for male bars.
    femaleColor string "#ec4899" Color for female bars.
    showPercentage boolean true Show percentage alongside counts.
    domainPadding number 0.1 Buffer percentage for chart edges.
    xAxisFormat 'count' \| 'percent' 'count' X-axis format.
    compact boolean - Compact mode for sidebar.

    Example

    {
  "type": "populationpyramid",
  "title": "Demographics",
  "dataSource": "population",
  "options": {
    "genderField": "KON",
    "ageField": "ALDER",
    "maleValue": "M",
    "femaleValue": "K",
    "maxAge": 90,
    "xAxisFormat": "percent"
  }
}
